Projects

Bidhub

BidHub is an app created with React and Express in order to help developers create collaborative coding projects. This was itself an organized collaborative work, created together with the help of two other developers. Users can create projects themselves and request admission to projects created by other users. Once users have agreed to work together, real time communication is enabled using WebSockets.

Automated Personal Wiki

A personal wiki app that I created because I found myself annoyed with existing software. I found that needing to take an extra step to create links, even if in a simple format, interfered with flow. So this app will automatically create links based on the page titles: both new mentions of old titles and previous mentions of new page titles will automatically result in links.
